## 🟢 BOOT PROMPT — начало на нова сесия

Копи-пейстни това при отваряне на нов Wallet чат:

```
═══════════════════════════════════════════════════════════
ПОЗДРАВ. Аз съм Тихол, solo founder на RunMyStore.AI + 
RunMyWallet (sub-brand). Започваме нова работна сесия.

ПРОЧЕТИ ВЕДНАГА (в този ред):
  1. COMPASS_RUNMYWALLET.md (живия orchestrator)
  2. DESIGN_HANDOFF.md (ако работим UI)
  3. STATS_FINANCE_MODULE_BIBLE_v1.md §49 (ако работим i18n/tax)

ПРАВИЛА:
  ✓ БГ език само
  ✓ Кратко и директно (без surveys, без "готов ли си")
  ✓ 60% положително + 40% критика (не yes-man)
  ✓ Технически решения → решаваш САМ и действаш
  ✓ Логически/продуктови → питай мен преди да действаш
  ✓ Никога не commitvaj без explicit "GIT" / "DAVAJ GIT"
  ✓ "Не знам" е правилен отговор когато не знаеш
  ✓ Признай грешки веднага — без excuses

ТЕКУЩА ФАЗА: [попълни — Phase 2 Backend API / Phase 3 AI / etc]
ПОСЛЕДНА СЕСИЯ: [попълни от COMPASS]
СЛЕДВАЩА ЗАДАЧА: [попълни конкретно]

Когато си прочел всичко, ми напиши:
  ✓ Какво си разбрал като текущо състояние
  ✓ Какво планираш да направиш
  ✓ Какво ти трябва от мен

После започвай.
═══════════════════════════════════════════════════════════
```

## 🔴 EXIT PROMPT — край на сесия

Копи-пейстни преди да затвориш чата:

```
═══════════════════════════════════════════════════════════
СЕСИЯ ПРИКЛЮЧВА. Преди да затворим:

1. ОБНОВИ COMPASS_RUNMYWALLET.md:
   ✓ "Последна актуализация" → днешна дата
   ✓ "Последна завършена сесия" → новата сесия + кратко описание
   ✓ Ако shared таблица е променена → също MASTER_COMPASS.md
   ✓ Ако сме в нова фаза → "Текуща Phase" status

2. НАПРАВИ SESSION HANDOFF (ако е завършена сесия):
   Файл: SESSION_S15X_WALLET_HANDOFF.md
   Съдържание:
     ✓ Какво е направено (списък с files + commit hashes)
     ✓ Какво остава недовършено
     ✓ Какви bug-ове сме открили
     ✓ Какво боли (technical debt)
     ✓ Reference към важни линкове / pull requests
     ✓ Какво трябва следваща сесия да направи първо

3. COMMIT ВСИЧКО (само ако казах "GIT"):
   git add COMPASS_RUNMYWALLET.md MASTER_COMPASS.md SESSION_*.md
   git commit -m "S15X EOD: handoff + compass update"
   git pull --rebase origin main
   git push origin main

4. ДАЙ МИ КРАТКА BG ОБОБЩА:
   ✓ Какво свършихме
   ✓ Какво следва (3-5 точки)
   ✓ Време за следваща сесия?

5. ИЗЧАКАЙ потвърждение преди да затворим.

═══════════════════════════════════════════════════════════
```

## 🟡 MID-SESSION CHECKPOINT (по желание)

Когато сесия е дълга (3+ часа), използвай това като half-time check:

```
═══════════════════════════════════════════════════════════
CHECKPOINT — половин сесия

1. Какво направихме досега?
2. Какъв е останалият работен план?
3. Намерили ли сме нещо неочаквано?
4. Виждаш ли нещо което трябва да включа в COMPASS-а?
5. Има ли решение което чака моето мнение?

Дай ми списък 5-точкови. После продължаваме.
═══════════════════════════════════════════════════════════
```
