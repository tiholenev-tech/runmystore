## 🚨 ESCALATION RULES

Ако сценарий fail-не:

| Брой нощи подред fail | Действие |
|---|---|
| 1 нощ | Шеф чат пише в STRESS_BOARD графа 4 (открити бъгове) |
| 2 нощи | Шеф чат повтаря в MORNING_REPORT — приоритет P0 |
| 3 нощи | Шеф чат прави retrospective — възможни причини: Code чат не разбра, фиксът е грешен, mimic bug |
| 5 нощи | Тихол + Шеф чат + Code чат прави meeting — ескалация |
| 7 нощи | Сценарий маркиран като „blocker" → влиза в beta blocker списъка |
